My Tech Journey

So anyone who knows me, knows how my tech journey has been like a rollercoaster. There are moments where I have felt I'm on track and headed towards a successful career and there are times I wonder what I'm doing. Over the last few months I've been trying to get back on track and focus on one thing, because in the past I've been know to jump ship from on track to another, I've done front end one day and the next day I'm learning Kotlin. But they say jack of all trades, master of none. So technically I've decided to focus on one thing, to become a master of one, everytime I jumped ships I've always come back to one thing and that is Front End Web development.

So this whole blog feels like I'm just rambling, but I'm getting to the point. I came to know about HNG internship a year ago, and I've been waiting for the cohort announcement. So when I joined the Zuri Training and was on their mailing list, I was sent an email to enroll for the internship https://internship.zuri.team.

Here are my goals for the internship:

  • Stay motivated throughout the internship.
  • Deliver timely and well done work
  • Continue learning and getting better
  • Network

To anyone feeling like they have tried so many times to get up but they are still falling like me, don't be ashamed to restart, remember to write your goals down on what you wish to achieve and work towards them. Hold yourself accountable and push yourself to become better. This blog is my accountability partner and I'll keep writing more articles on it on the progress I've made so far.

Before I end the blog, if you're starting out your tech journey, here are some links that might be of help to you.

  • UX/UI design using figma, tutorial for beginners - youtube.com/watch?v=jk1T0CdLxwU
  • You'll need to learn how to use git, so here is git tutorial for beginners - https://www.youtube.com/watch?v=HVsySz-h9r4
  • If you're looking to get into web programming then start learning the fundamentals of web programming starting with html - https://www.w3schools.com/html/default.asp
  • And for those looking to jump into Mobile development, Android development is a great place to start, kotlin also happens to be a fast growing language that is preferred for android. Here's the link to where you can start -https://developer.android.com/kotlin/learn
